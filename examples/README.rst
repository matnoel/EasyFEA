.. _easyfea-examples:

Examples
========

Contains example scripts demonstrating the use of numerous simulations that can be run with EasyFEA.

New to EasyFEA? Start with the :ref:`HelloWorld` script described in the :ref:`begin`.

.. list-table::
    :header-rows: 1

    * - Folder
      - Description
    * - :ref:`Beam <easyfea-examples-beam>`
      - Euler-Bernoulli beam simulations (1D/2D/3D).
    * - :ref:`DIC <easyfea-examples-dic>`
      - Digital Image Correlation analyses.
    * - :ref:`LinearizedElasticity <easyfea-examples-linearized-elasticity>`
      - Elastostatic, dynamic, contact, homogenization, and mesh optimization.
    * - :ref:`Hyperelasticity <easyfea-examples-hyperelasticity>`
      - Nonlinear hyperelastic static and dynamic simulations.
    * - :ref:`Thermal <easyfea-examples-thermal>`
      - Static and transient thermal simulations.
    * - :ref:`PhaseField <easyfea-examples-phasefield>`
      - Quasi-static brittle fracture via phase-field damage models.
    * - :ref:`WeakForms <easyfea-examples-weak-forms>`
      - Static, transient, and dynamic simulations (linear & nonlinear).
    * - :ref:`Meshes <easyfea-examples-meshes>`
      - Mesh creation examples (2D and 3D geometries).
